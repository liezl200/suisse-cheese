# suisse-cheese
A companies research platform for Code Suisse, complete with smart charts and machine learning-based analysis

# technology stack
Node.js
Express.js
Knexjs (a query builder for SQL)
MySQL database

# setup
cd suisse-cheese && sudo npm install
